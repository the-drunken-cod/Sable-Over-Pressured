## Sable: Over-Pressured
A datapack that adds air pressure above the Nether roof.

</div>
<br>

Changes Sable's air pressure curve for above the Nether roof, making it possible to build planes, airships and other air-propelled craft up there.  
The pressure curve follows that of The End closely, so expect the same kind of physics.
  
<br>

### Installation
1. Download the latest release from the [GitHub releases page](https://github.com/the-drunken-cod/Sable-Over-Pressured/releases) or the [Modrinth page.](https://modrinth.com/datapack/sable-over-pressured)
2. Place the downloaded `.zip` file into your world's `datapacks` folder.
3. Reload the datapack using `/reload` in the game or by restarting your world or server.
